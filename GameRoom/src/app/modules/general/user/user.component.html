Instructions:

Each player must register with a real or fictitious account in the game room.

This account will be used to record the scores obtained in each game and determine the final results.

Every player will participate in the four games available in the room.

After completing each game, the points obtained will be recorded in the results tab.

Each game will have a Top 5 based on the performance of the players.

Points will be assigned according to each player's position in the Top:

  • Top 1: 5 points
  • Top 2: 3 points
  • Top 3: 1 point
  • Top 4: 0 points
  • Top 5: Can subtract one point from another player.

If a player appears in the Top of a game more than once, only the highest score will be counted.

At the end of the stipulated time, the composition of the Top for each game will be verified.

The points obtained by each player in each game will be summed up to determine the overall performance.

The player with the highest total points will be the winner.
